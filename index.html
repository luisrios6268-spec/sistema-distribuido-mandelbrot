🌀 Mandelbrot Distribuido

Proyecto con Rust + Docker + ZeroTier

Descripción Arquitectura Tecnologías Ejecución

📌 Descripción

Este proyecto implementa un sistema distribuido para generar el conjunto de Mandelbrot. Utiliza un modelo coordinador-workers donde múltiples nodos colaboran para procesar bloques de la imagen.

🧠 Arquitectura

  • Coordinador: Divide el trabajo en bloques
  • Workers: Procesan cada bloque
  • Red: Comunicación mediante ZeroTier

⚙️ Tecnologías

  • 🦀 Rust
  • 🐳 Docker
  • 🌐 ZeroTier
  • 🔧 Git & GitHub

🚀 Cómo ejecutar

Clona el repositorio:

git clone https://github.com/luisrios6268-spec/sistema-distribuido-mandelbrot.git

Levanta los contenedores:

docker-compose up

Ejecuta workers:

cargo run
